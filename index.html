Dashboard

M-Dashboard

Comprehensive dashboard covering Financing, Funding, Hajj, QRIS, MDIN, and NPF, including MoM and YTD comparisons.

Feature:
1. Filter by Co-Branch, Branch, Product & Group
2. Select active month
3. Pick theme between dark and light
4. Select & Unselect legend to filter chart item
5. Pick tab between Financing, Funding, Hajj, MDIN, QRIS & NPF

M-Leading Indicator

Monthly indicator dashboard for the growth of CA, SA, Funding, Consumer Bookings, SME Bookings, Hajj Quota, MDIN, QRIS, and New CIF versus monthly targets.

Feature:
1. Filter by Co-Branch, Branch
2. Select active month
3. Pick theme between dark and light
4. Display Ranking Performance (Branch or Co-Branch)
5. Table contains detailed data breakdown on Web version

Consumer Leads

M-Leads

Real-time monitoring of total sales team, leads injected by HO, follow-up, interest, and booking conversion rate.

Feature:
1. Filter by typing NIK/Name, or select Periods, Position or Branch or click on Marketing Name
2. Performance by Position & Top 5 Performers
3. Pick theme between dark and light
4. Display Ranking Performance (Branch or Co-Branch)
5. Table contains detailed data breakdown

Agent Performance

M-Direct

Sales team access performance, featuring daily login rate monitoring, total active agents, and monthly activity tracking.

Feature:
1. Filter by typing NIK/Name, or select Periods, Position, Branch and MTD Login Status or click on Marketing Name
2. Performance Total Agents, Login MTD, Not Login MTD, Login Rate Convertion
3. Pick theme between dark and light
4. Display Chart Monthly Progress M-Direct by clicking Marketing Name
5. Table contains detailed data breakdown
